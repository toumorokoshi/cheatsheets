Useful Commands
===============

* $ sed -i s/SEARCH_PATTERN/REPLACE_PATTERN/g FILE_NAME: replace SEARCH_PATTERN with REPLACE_PATTERN is FILE_NAME
* $ git ls-files | xargs cat | wc -l : count number of lines in a git repository.
