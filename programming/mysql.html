Common Commands

  • Show all mysql users: SELECT user FROM mysql.user;
  • Show grants for a user: SHOW grants for 'USER'@'DOMAIN';
  • Grant a user all privileges to a db: GRANT ALL PRIVILEGES ON 'DB'.* TO 'USER'@'DOMAIN'
